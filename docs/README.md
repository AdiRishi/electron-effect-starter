# Docs

Durable documentation for this repository. Code comments explain a file; the documents here explain the system.

- **[`adr/`](./adr/README.md)** — Architecture Decision Records: why the hard-to-reverse decisions were made. Start here to understand anything in the codebase that looks surprising.
- **`plans/`** — implementation plans for coordinated initiatives (created when the first plan is needed; see the ADR README for the spirit of the split: plans say _what we intend to build_, ADRs say _why a decision was made_).
- **[`../.repos/`](../.repos/README.md)** — vendored read-only reference repositories (Effect source pinned to the installed version), for agents and humans to learn idiomatic patterns from real source.

Top-level project documentation (README, agent guidance) is maintained separately and intentionally minimal for now.
